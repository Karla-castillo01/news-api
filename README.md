# Welcome!
News-Api is a Python-based web application built with Flask that integrates a News API to fetch and display top articles based on user search queries. The project demonstrates API integration, dynamic content rendering, and automated testing for reliability.

## Features
Search Functionality: Users can enter keywords to retrieve relevant news articles.

Top Articles Display: Results are shown in a clean, responsive webpage.

API Integration: Uses News API for real-time news data.

Testing Included: Unit tests ensure API calls and rendering logic work as expected.

## Tech Stack
Backend: Python, Flask

Frontend: HTML, CSS (via Flask templates)

API: https://newsapi.org/

Testing: unittest or pytest (depending on your implementation)
